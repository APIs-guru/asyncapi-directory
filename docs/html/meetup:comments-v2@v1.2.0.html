Meetup Comments v2 (stream) 1.2.0

This is a streaming API that has been autogenerated from the Meetup using Streamdata.io.

Connection details

URL Scheme Description
api.meetup.com/2/event_comments		 https
URL Variables
Name Default value Possible values Description
comment_id None Any

Return comments for a given set of comment IDs, separated by commas
member_id None Any

Return comments for the given member_ids, separated by commas
fields None Any

Optionally accepts the value "member_photo" or "notifications"
group_id None Any

Return comments in groups with these ID numbers, separated by commas
event_id None Any

Return comments on these events, separated by commas.
callback None Any

Name of a function to be called with an array of Event Comment notification objects. If this parameter is not supplied, the chunked stream is joined instead.
since_mtime None Any

Should be supplied for all but the first polling request, so that any missed notifications are can be sent in an immediate response
since_count None Any

Request that some number of recent messages be sent immediately, if available. May not be specified in the same request as since_mtime.
api_version None Any
